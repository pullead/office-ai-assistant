# QA Stability Checklist

このチェックリストは、主要機能の回帰確認に使うための一覧です。  
大きな修正のあとに順番に確認することを推奨します。

---

## 共通

### 起動

- [ ] 起動時にエラーダイアログが出ない
- [ ] 左ナビのアイコンがずれない
- [ ] 左ナビで複数項目が同時選択されない
- [ ] メイン領域が空白にならない

### 言語切替

- [ ] 日本語で正常表示される
- [ ] 英語へ切替後も UI が消えない
- [ ] 中国語へ切替後も UI が消えない
- [ ] ラベル / ボタン / プレースホルダが切り替わる

### レイアウト

- [ ] 横幅変更で重なりが出ない
- [ ] 高さ変更で操作欄が消えない
- [ ] スクロール領域が最後まで見える

### エラー処理

- [ ] エラー時にアプリ全体が落ちない
- [ ] エラー文が読める
- [ ] 右レポートまたはログ欄に情報が残る
