# Bug Tracking

最終更新日: 2026-05-01

このファイルでは、安定化フェーズで確認した問題を優先度順に管理します。

---

## 優先度ルール

- `P0`: 起動不能、画面消失、保存不能などの致命障害
- `P1`: 主導線が止まる、または実用上の影響が大きい問題
- `P2`: 動作はするが、表現や精度に改善余地が大きい問題
- `P3`: 見た目、文言、将来改善向けの項目

## ステータス

- `open`
- `investigating`
- `fixing`
- `verifying`
- `closed`
