# Known Issues

現時点で把握している制約と、今後の改善候補をまとめています。

## 1. OCR 精度

- 画像品質が低い場合は、金額や日付の抽出精度が不安定になることがある
- 合成帳票での基礎回帰は自動化済みだが、実帳票サンプルはまだ十分ではない
- 非定型文書では、キー / 値候補の抽出が中心であり、完全な構造化までは保証しない

## 2. データ可視化

- `TXT` 入力は棒 / 線 / 円グラフへ対応したが、内容は頻出語ベースの簡易集計となる
- 高度な列推定や複雑な時系列解釈は、現時点では `CSV / Excel` の方が安定している

## 3. Web 抽出

- JavaScript 依存の強いページは本文抽出が不安定になる場合がある
- サイト構造によっては、本文よりもナビゲーション文言を拾う場合がある

## 4. AI API

- 外部 API の仕様変更や一時障害により、接続診断や要約に失敗する場合がある
- API 側のレート制限や認証エラーは、アプリ側では警告表示で継続する

## 5. ファイル整理

- アーカイブ閲覧は現時点では `ZIP / TAR 系` を中心に対応している
- `7z / rar` などの追加形式は今後の段階拡張候補
- 大規模フォルダの Space Lens 解析は、環境によって初回スキャンに時間がかかる

## 6. 表示と国際化

- 一部のコンソール出力では、PowerShell 側の文字コード設定によって文字化けして見える場合がある
- 画面内の主要文言は多言語切り替えに対応しているが、今後さらに細かな文言の統一余地がある
